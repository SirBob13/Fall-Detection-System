# Admin dashboard

Next.js admin UI for the fall-detection system (users, devices, overview).

## Setup

```bash
cd admin-dashboard
npm install
```

## Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Point the app at your FastAPI backend URL via the project’s env / config (same API the mobile app uses).

## Build

```bash
npm run build
npm start
```
